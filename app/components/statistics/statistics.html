По данным ВОЗ каждый третий человек заражен гельминтами

Согласно данным за 2019 год, более 1,5 млрд человек заражены гельминтами, которые передаются через почву.

Количество человек, зараженных гельминтами, которые передаются
другим путем увеличивается в несколько раз.

*Информация взята из открытых источников ВОЗ
